# 🎂 Birthday Game

A fun and interactive **3D web-based birthday celebration** built with **Three.js**. Step into a virtual birthday setup where you can celebrate your special day by cutting a cake placed in front of you and playing your favorite songs through an in-game phone.

## ✨ Features

- 🎂 Interactive birthday cake
- 🔪 Cut the cake with simple interactions
- 📱 Play songs using an in-game phone
- 🎵 Background music support
- 🌟 Immersive 3D environment
- 💻 Runs directly in the browser
- ⚡ Built with Three.js for smooth WebGL rendering

## 🛠️ Tech Stack

- **Three.js**
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **WebGL**

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/NostalgicWinters/BirthdayGame.git
```

### Navigate to the project

```bash
cd BirthdayGame
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open the local URL shown in your terminal to play.

## 🎮 How to Play

1. Load into the birthday room.
2. Look at the birthday cake placed in front of you.
3. Interact with the knife to cut the cake.
4. Use the phone to browse and play available songs.
5. Enjoy the virtual birthday celebration!
